Button

A button allows user to make choices, and triggers event based on the action.

Contained Button

Contained buttons have more emphasis, and they are used as a primary action to your app.

Outline Button

Outlined buttons are used for action which have lower emphasis than primary button.

Round Button

You can also have rounded corners for your buttons by adding the btn-round class.

Icon Button

Icon buttons are icon and text combination.

Floating Action Button

Floating action button represents primary action of a screen.

Link Button

Link button are used to primarily represent links.
