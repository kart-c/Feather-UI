Avatar

Avatars are user profile pictures, usually found in circular shapes. Avatars can be used for 'container' objects — projects, spaces, repositories, etcetera — to give them a recognisable visual identity.

Profile Avatar

To create profile avatars simply add avatar class to your img tag.

Avatar Sizes

You can also have different sized square avatars by adding avatar-lg, avatar-md or avatar-sm as a class.
