Image

Use our image component to style your images.

Square Image

Add the sq-img class to make any image into a square shaped image.

Round Images

Add the round-img class to make circular shaped images.

Responsive Images

Use the resp-img class to make your images responsive. Add a parent element for sizing purposes.

Note : If the image position is not as per your requirement, use the object-position property to position the image.
